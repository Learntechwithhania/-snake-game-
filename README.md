The Ultimate Snake Experience with stunning neon visuals and smooth gameplay!

## 🎮 About
A modern, feature-rich snake game with:
- 🎨 Multiple themes (Neon, Retro, Nature)
- 🐍 Different snake skins (Fire, Ice, Rainbow)
- ⚡ Power-ups and special abilities
- 🏆 High score tracking
- 📱 Mobile-friendly controls
- 🔊 Sound effects and music

## 🚀 Play Now
Simply open `index.html` in your browser to start playing!

## 🎯 Features
- Smooth animations and particle effects
- Multiple difficulty levels
- Wall modes (Die on wall / Wrap around)
- Combo system for bonus points
- Golden food and power-ups
- Lives system
- Encouraging messages

## 🕹️ Controls
- **Arrow Keys** or **WASD**: Move snake
- **Space**: Pause/Resume
- **Mobile**: Touch controls and swipe gestures
